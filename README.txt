CashBackHacker — автоматизация тестирования
Проект для домашнего задания курса по автоматизации тестирования.

Что здесь
Сервис CashbackHackService с методом remain(int amount) для подсчёта остатка до следующей тысячи.

Автоматические тесты на трёх фреймворках:

TestNG (ветка testng)

JUnit4 (ветка junit4)

JUnit5 + поддержка legacy JUnit4 (ветка junit4-platform)

Ветка main — исходный проект с сервисом.

Настроен Gradle для сборки и запуска тестов.

В репозитории есть issue с баг-репортами и отчётами по автотестам.
